Declutter your Discord server

Needle is a bot that helps you take full advantage of Discord Threads by grouping relevant content consistently and automatically.

  • 100% free
  • 24/7 uptime
  • open source
  • highly customizeable
  • no data collection
  • runs on renewable energy
Add to server Learn more

Features

  • Automatic thread creation

    Easily create ticketing systems, discussion boards, and support channels tailored to your server. New threads are created for each new message in a given channel.

    Configuration

    /configure autothreading <channel> <enabled>
