Features

  • Create threads automatically

    Easily create ticketing systems, discussion boards, and support channels tailored to your server. New threads are created for each new message in a given channel.

    Configuration

    /auto-thread

  • Full control

    Thread creators can change title and archive threads without any special permissions. Threads are archived in a way that lets anyone un-archive it in the future by default.

    Commands

    /title <new title>

    /close

  • Message customization

    The output messages are customizable, so that the bot can fit the theme of your french medieval meme server.

    Configuration

    /setting

  • Help and support

    Forgotten all the commands shown above already? Use the help command to see all the available commands as well as how to use them. If you still have questions you are more than welcome to join the support server!
