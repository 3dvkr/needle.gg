Features

  • Automatic thread creation

    Easily create ticketing systems, discussion boards, and support channels tailored to your server. New threads are created for each new message in a given channel.

    Configuration

    /configure autothreading <channel> <enabled>

  • Full control

    Thread creators can change title and archive threads without any special permissions. Threads are archived in a way that lets anyone un-archive it in the future by default.

    Commands

    /title <new title>

    /close

  • Message customization

    Most of the output messages are customizeable, so that the bot can fit the theme of your french medieval meme server.

    Configuration

    /configure message <message key> <value>

  • Help and support

    Forgotten all the commands shown above already? Use the help command to see all the available commands as well as how to use them. If you still have questions you can join the support server.

    Commands

    /help

    /help <command>
